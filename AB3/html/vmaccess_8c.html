static void vmem_put_page_into_mem ( int  address)

This function puts a page into memory (if required). Ref Bit of page table entry will be updated. If the time window handle by g_count has reached, the window window message will be send to the memory manager. To keep conform with this log files, g_count must be increased before the time window will be checked. vmem_read and vmem_write call this function.

Variable Documentation

int g_count = 0
static

global acces counter as quasi-timestamp - will be increment by each memory access

The progression of time is simulated by the counter g_count, which is incremented by vmaccess on each memory access. The memory manager will be informed by a command, whenever a fixed period of time has passed. Hence the memory manager must be informed, whenever g_count % TIME_WINDOW == 0. Based on this information, memory manager will update aging information
