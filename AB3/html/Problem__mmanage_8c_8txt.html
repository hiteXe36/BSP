int find_unused_frame ( )
static

This function finds an unused frame.

The frameinfo array marks unused frames with VOID_IDX. Based on this information this function searchs in frameinfo for the free frame with the smallest frame number.

Returns
idx of the unused frame with the smallest idx. If all frames are in use, VOID_IDX will be returned.
